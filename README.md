# txxxt

ASCII video in your terminal.

> Work in progress

## Build

```bash
cargo build --release
./target/release/txxxt
```

## Keybindings

| Key | Action |
|-----|--------|
| `o` | Toggle outline mode |
| `c` | Toggle color |
| `v` | Cycle character set |
| `+`/`-` | Adjust brightness threshold |
| `q` / `ESC` | Quit |
